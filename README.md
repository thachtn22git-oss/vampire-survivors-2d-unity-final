# 🎮 Vampire Survivors 2D (Unity)

A **2D action survival game** inspired by *Vampire Survivors*, developed with **Unity** and **C#**.

The objective is simple: survive as long as possible while defeating endless waves of enemies, collecting experience, leveling up, and unlocking powerful upgrades.

---

## 📌 Overview

This project was developed to practice:

- Unity 2D game development
- C# programming
- Object-Oriented Programming (OOP)
- Gameplay programming
- Enemy AI
- UI implementation
- Game architecture and debugging

---

## ✨ Features

### 🎮 Gameplay
- Player movement
- Auto attack system
- Endless enemy waves
- Enemy AI behavior
- Experience and level-up system
- Upgrade selection
- Score & High Score system

### ⚔ Weapons
- Directional weapons
- Area-of-effect weapons
- Multiple attack behaviors

### 🖥 User Interface
- Main Menu
- In-game HUD
- Pause Menu
- Game Over Screen
- Level Up Panel
