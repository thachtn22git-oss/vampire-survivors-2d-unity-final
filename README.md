# Vampire Survivors – 2D (Unity)

## Description
A 2D survival game inspired by Vampire Survivors, developed using Unity.
Players control a warrior character and survive against endless waves of enemies.

## Features
- Player movement and auto-attack system
- Multiple weapon types (Directional & Area)
- Enemy AI and spawning system
- Experience, level-up, and upgrade mechanics
- Point system and best score saving
- UI system with pause and game over screens

## Technologies
- Unity 6.2
- C#
- 2D URP
